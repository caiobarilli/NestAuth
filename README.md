| app         | host      | port |

### Pré-requisitos

#

Para começar, certifique-se de ter o [Docker](https://docs.docker.com/) e [Docker Compose](https://docs.docker.com/compose/install/) instalado no seu sistema.


### Download

#

git clone git@github.com:caiobarilli/NestAuth.git
